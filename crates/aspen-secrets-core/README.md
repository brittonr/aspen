# aspen-secrets-core

Portable secrets engine contracts, state, DTOs, and limits for Aspen.

This crate is the dependency-light home for KV, Transit, and PKI request/response models plus durable state shapes shared by Aspen secrets runtime crates and downstream consumers. It contains no runtime storage, transport, handler, SOPS, or authentication-service implementation.
